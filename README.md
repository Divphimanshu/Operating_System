# 💻 Operating_System_Algorithms

Welcome to **Operating_System_Algorithms**!

This repository contains implementations of fundamental **Operating System algorithms and concepts** using C. It serves as a practical learning resource for understanding core OS topics through code.

---

## 📚 Topics Covered

- FCFS (First Come First Serve)
- Shortest Job First (SJF)
- Round Robin Scheduling
- Disk Scheduling Algorithms
- Page Replacement Algorithms
- Banker's Algorithm
- Process Control Block (PCB)
- Segmentation
- Producer-Consumer Problem
- Reader-Writer Problem
- Dining Philosophers Problem
- Simple Shell

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Divphimanshu/Operating_System_Algorithms.git
```

### Navigate to the repository

```bash
cd Operating_System_Algorithms
```

### Compile a program

```bash
gcc filename.c -o output
```

### Run the program

```bash
./output
```

**Windows (MinGW):**

```bash
gcc filename.c -o output.exe
output.exe
```
